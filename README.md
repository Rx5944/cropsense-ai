# CropSense AI

AI-powered crop recommendation and profitability prediction system.

## Features

- Predict best crop using soil + climate data
- Top 5 suitable crops
- Profitability ranking
- Smart AI explanation

## Tech Stack

- Python
- Scikit-learn
- Streamlit
- Pandas

## Run

pip install -r requirements.txt

python model/train.py

streamlit run app.py# cropsense-ai
